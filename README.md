# PRODIGY_WD_03

A responsive Tic-Tac-Toe web application featuring interactive gameplay with real-time win detection and smooth user interaction. Developed using HTML, CSS, and JavaScript as part of Prodigy InfoTech Web Development Internship (Task-03).

This project is a Tic-Tac-Toe game developed as Task-03 of the Prodigy InfoTech Web Development Internship. It allows users to play the classic Tic-Tac-Toe either against another player or an AI opponent. The application dynamically tracks the game state, checks for winning combinations, and ensures a smooth and engaging experience across all devices.

🚀 Features
Interactive Tic-Tac-Toe gameplay
Player vs Player mode
Player vs AI mode
Real-time win and draw detection
Game reset functionality
Fully responsive design (mobile, tablet & desktop)
Clean and modern UI

🛠️ Technologies Used
HTML5 – Structure of the game layout
CSS3 – Styling, layout, and responsiveness
JavaScript – Game logic, DOM manipulation, and win detection
